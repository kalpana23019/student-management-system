# Student Management System

## Project Description

A web-based Student Management System built using Java, JSP, Servlets, JDBC, PostgreSQL, and Apache Tomcat.

## Features

* Add Student
* View Students
* Update Student
* Delete Student
* Search Students
* Upload Student Photo
* PostgreSQL Database Integration

## Technologies Used

* Java
* JSP
* Servlets
* JDBC
* PostgreSQL
* Apache Tomcat
* Bootstrap
* Git & GitHub

## Project Structure

StudentManagementSystem
├── controller
├── dao
├── daoImpl
├── model
├── util
├── jsp
├── images
└── css

## Database

Database: student_db

Table: students

Columns:

* student_id
* first_name
* last_name
* email
* phone
* gender
* course
* address
* photo

## How to Run

1. Install PostgreSQL
2. Create database student_db
3. Update DBConnection credentials
4. Configure Apache Tomcat
5. Deploy project
6. Open in browser
